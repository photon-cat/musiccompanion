# Aria - AI Music Companion

An interactive AI music companion featuring a 3D anime avatar that dances to music in real-time, tracks your facial expressions, and chats with you via text or voice.

![Built with Next.js](https://img.shields.io/badge/Next.js-15-black) ![Python](https://img.shields.io/badge/Python-FastAPI-blue) ![Three.js](https://img.shields.io/badge/Three.js-VRM-green) ![Gemini](https://img.shields.io/badge/Google-Gemini_2.5-orange)

## Features

- **Music-Synced Dancing** - Avatar performs choreographed dances synchronized to beat, energy, and mood of the music. Songs are pre-analyzed with librosa and choreography is generated by Gemini.
- **3D VRM Avatar** - Anime-style avatar rendered with Three.js, supporting blend shape expressions, skeletal animations (VRMA), and procedural dance motions (hip bounce, arm swings, chest twists, etc).
- **Real-Time Face Tracking** - MediaPipe captures your head pose, smile, blinks, and brow movements, and mirrors them onto the avatar.
- **Voice Chat** - Speak to Aria in real-time via Gemini Live API with 26+ voice options. She responds with synthesized speech and animated reactions.
- **Text Chat with Tool Use** - Gemini-powered conversation with function calling to trigger animations, change expressions, and play music.

## Getting Started

### Prerequisites

- Node.js 18+
- Python 3.11+
- Google Gemini API key

### Setup

```bash
# Install frontend dependencies
npm install

# Install backend dependencies
pip install fastapi uvicorn google-genai librosa pyyaml

# Set your API key
export GEMINI_API_KEY=your_key_here
```

### Run

```bash
# Start the backend
python app.py

# Start the frontend (in another terminal)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start chatting with Aria.

## Music Pipeline

Songs go through a preprocessing step that extracts audio features and generates choreography:

1. **Feature extraction** - librosa analyzes BPM, RMS energy, spectral centroid, chroma, onset strength, and valence in 4-second windows
2. **Choreography generation** - Features are sent to Gemini which produces timestamped dance directives (poses, expressions, animations)
3. **Playback** - The frontend reads the choreography script and triggers avatar motions in sync with audio playback

Pre-choreographed songs are included in `public/music/`.
